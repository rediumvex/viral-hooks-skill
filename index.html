Hook Library — General-purpose tool

Top 100 Viral Hook Formulas

Battle-tested hook structures for short-form video, posts, threads, and headlines. Organized by 10 psychology triggers, 100 formulas, with examples and best-platform tags. Bottom of the page has the flat copy-paste list for quickly handing off to a writer alongside a script.

How to use this library

  1. Read the script first — figure out the topic and emotional tone.
  2. Pick a category from the menu below that matches the script's intent.
  3. Scan the 10 formulas in that category and pick 3 that fit.
  4. Fill in the brackets with the script's specific topic / number / pain.
  5. Test all 3 as alt-hooks (A/B testing reveals which works for the audience).

Workflow tip. Send this file together with the finished script. The writer reads the script first, identifies the emotional beat (curiosity / contrarian / confession / data / story), jumps to the matching category, picks 3 candidate hooks, fills the brackets with script-specific words, and tests them as A/B variants.

The two best hooks per video are usually different categories. Mix one stop-scroll hook (negation, specificity, question) with one retention hook (story, confession, emotional). The first stops the thumb. The second keeps them watching past 3 seconds.
